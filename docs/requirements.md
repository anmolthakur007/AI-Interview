\# AI Interview Platform



\## Goal



Build an AI-powered interview platform that helps users practice technical interviews.



\---



\## Candidate Features



\- Register

\- Login

\- Upload Resume

\- Start Interview

\- Answer Questions

\- Receive AI Feedback

\- View Previous Interviews



\---



\## Recruiter Features



\- Login

\- Create Interview

\- View Candidate Reports



\---



\## AI Features



\- Resume Analysis

\- Question Generation

\- Answer Evaluation

\- Feedback Generation
